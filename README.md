# 🤖 Machine-Learning

A collection of machine learning projects exploring real-world datasets for predictive analysis.

---

## 📁 Projects

### 🫀 Heart Disease Prediction
**Files:** `Heart.ipynb` · `heart.csv`

Predicts the likelihood of heart disease in a patient based on clinical features such as age, cholesterol levels, blood pressure, and more.

- Exploratory Data Analysis (EDA)
- Data preprocessing & feature engineering
- Classification model training & evaluation
- Metrics: Accuracy, Precision, Recall, F1-Score

---

### 💰 Insurance Cost Prediction
**Files:** `Insurance.ipynb` · `insurance.csv`

Estimates medical insurance charges based on demographic and lifestyle factors like age, BMI, smoking status, and region.

- Exploratory Data Analysis (EDA)
- Feature encoding & data preparation
- Regression model training & evaluation
- Metrics: RMSE, MAE, R² Score

---

## 🛠️ Tech Stack

- **Language:** Python 3
- **Environment:** Jupyter Notebook
- **Libraries:** NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn

---

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/sakshidangi2006/Machine-Learning.git
   cd Machine-Learning
   ```

2. **Install dependencies**
   ```bash
   pip install numpy pandas matplotlib seaborn scikit-learn jupyter
   ```

3. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

4. Open either `Heart.ipynb` or `Insurance.ipynb` and run the cells.
